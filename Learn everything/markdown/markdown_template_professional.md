## <img src="https://img.icons8.com/ios-filled/16/000000/info.png"/> Introduction
This is a **professional Markdown template** with badges and HTML-based icons, suitable for **README.md** files or internal documentation.

---

## <img src="https://img.icons8.com/ios-filled/16/000000/star.png"/> Features
- Professional look with **Shields.io badges**
- HTML icons for clarity
- Easy to copy & reuse

## <img src="https://img.icons8.com/ios-filled/16/000000/download.png"/> Installation
```bash
# Clone repository
git clone https://github.com/your-repo/project.git

# Install dependencies
npm install
```

## <img src="https://img.icons8.com/ios-filled/16/000000/play.png"/> Usage
```bash
npm start
```

- Open `http://localhost:3000` in your browser.
- Default login: `admin / password`
