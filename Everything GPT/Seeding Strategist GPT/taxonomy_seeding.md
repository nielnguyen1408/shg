# Taxonomy seeding

## Cột `muc_dich`
| Nhãn | Mô tả |
|------|-------|
| kích hoạt thảo luận | Mở đầu, đặt câu hỏi, kéo người vào comment |
| xác nhận uy tín | Tăng độ tin cậy, củng cố thương hiệu |
| chia sẻ trải nghiệm | Mang tính review thật, tăng tính tự nhiên |
| phản biện nhẹ | Tạo cảm giác trung thực, có đối lập nhẹ |
| khuếch đại cảm xúc | Tăng năng lượng tích cực, lan tỏa cảm xúc |
| chốt hạ mua hàng | Dẫn dắt hành động, khuyến khích mua |
| định hướng thương hiệu | Hướng câu chuyện theo giá trị thương hiệu |
| hài hước / giải trí | Làm mạch thảo luận sinh động |

## Cột `chien_thuat`
| Nhãn | Mô tả |
|------|-------|
| hỏi gợi mở | Dùng câu hỏi để khơi gợi phản hồi |
| cảm thán tích cực | Dùng cảm xúc, emoji, dấu chấm than |
| so sánh nhẹ | Dẫn ví dụ đối chiếu nhẹ nhàng |
| chuyển ý tự nhiên | “Ban đầu… nhưng…” |
| truyền cảm hứng | Tạo năng lượng tốt, động viên |
| hài hước nhẹ | Giọng vui, dễ gần |
| đồng cảm cá nhân | Kể trải nghiệm thật |
| đề xuất / khuyến nghị | Mềm mại dẫn tới hành động |
